
🌦️ Automated Weather Dashboard
📌 Project Overview

The Automated Weather Dashboard is an end-to-end data analytics project that collects real-time weather data from a live API, processes it using Python, stores it in a SQL database, automates the data pipeline through Windows Task Scheduler, and visualizes insights in an interactive Power BI dashboard.

The project demonstrates the complete data engineering and business intelligence workflow, from data extraction to automated reporting.         

🚀 Project Workflow
1. Live Weather API
Connected to a live weather API to retrieve real-time weather information.
Collected data such as temperature, humidity, pressure, wind speed, weather conditions, visibility, sunrise, sunset, and location details.

2. Python Automation
Developed Python scripts to fetch data from the API.
Parsed JSON responses into structured datasets.
Cleaned, transformed, and validated the data before loading it into the database.

3. SQL Database
Designed a relational database schema to store weather information.
Created SQL tables for efficient data storage and retrieval.
Managed historical weather records using SQL.

4. Windows Task Scheduler
Automated the execution of Python scripts using Windows Task Scheduler.
Scheduled regular updates to ensure the database always contains the latest weather information.
Eliminated the need for manual data collection.

5. Power BI Dashboard
Connected Power BI to the SQL database.
Created fact and measure thable and star schema.
Measure table for more than 50+ DAX Queries.
Designed an interactive dashboard with KPIs, charts, filters, and maps.
Enabled automatic dashboard refresh whenever new data is added to the database.
